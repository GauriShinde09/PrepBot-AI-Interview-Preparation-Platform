# 🚀 PrepBot

### AI-Powered Interview Preparation Platform

PrepBot is a full-stack web application designed to help users prepare for technical and HR interviews through AI-powered mock interview sessions.

## 🔗 Live Demo

👉 https://your-live-demo-link.vercel.app

---

## ✨ Features

- AI-powered interview question generation using Grok AI
- Personalized technical and HR interview preparation
- Secure user authentication and authorization
- Performance tracking and interview analytics
- Interview history management
- Responsive and user-friendly interface
